# React Starter Kit

Includes:
- Vite + React + TypeScript
- Tailwind CSS (configured)
- Framer Motion
- A small set of reusable UI components (Button, Card, Navbar, Modal)

Quick start:
1. Unzip and `cd react-starter-kit`
2. Run `npm install`
3. Run `npm run dev`
4. Open http://localhost:5173

Notes:
- Tailwind needs `postcss` and `autoprefixer` (included in devDependencies).
- Adjust versions in package.json as needed.
